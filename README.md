# نقاء — تطبيق إدخال المنتجات 📦

تطبيق مستقل (منفصل عن تطبيقات العميل/السائق/الإدارة) مخصص فقط لإدخال منتجات جديدة بسرعة، مع ميزة **إزالة خلفية الصورة تلقائياً بالذكاء الاصطناعي** واستبدالها بخلفية بيضاء عليها علامة "نقاء" المائية — كل هذا يشتغل داخل الجهاز مباشرة (بدون أي اشتراك أو تكلفة شهرية، المكتبة مجانية بالكامل: `@imgly/background-removal`).

## ⚙️ الإعداد الأول (مرة واحدة)

1. أنشئ ملف `.env.local` بجذر المشروع فيه:
```
VITE_SUPABASE_URL=https://xxxxx.supabase.co
VITE_SUPABASE_ANON_KEY=xxxxxxxxxxxxx
```
(نفس القيم بالضبط المستخدمة بتطبيقات نقاء الثلاثة الأخرى — انسخها من أي واحد منهم).

2. ثبّت الحزم:
```bash
npm install
```

3. جرّب محلياً:
```bash
npm run dev
```

## 🚀 النشر (Vercel) — نفس طريقة التطبيقات الثلاثة

1. ارفع المجلد كمشروع جديد على GitHub.
2. اربطه بـ Vercel (نفس الحساب المستخدم للتطبيقات الأخرى).
3. أضف متغيرات البيئة (`VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`) بإعدادات المشروع على Vercel.
4. بعد أول نشر ناجح، Vercel بيعطيك رابط مثل: `naqaa-product-entry.vercel.app`.
5. **مهم:** افتح `capacitor.config.json` وحدّث قيمة `server.url` بالرابط الحقيقي اللي طلع لك (حالياً فيه رابط افتراضي مؤقت).

## 📱 تحويله لتطبيق أندرويد (APK) — نفس خطوات التطبيقات الثلاثة بالضبط

```bash
npm install -g @capacitor/cli   # مرة وحدة فقط إذا ما ثبتها قبل
npm run build
npx cap add android             # أول مرة فقط
npx cap sync android
npx cap open android            # يفتح Android Studio لبناء الـ APK
```

## 🖼️ كيف تشتغل ميزة إزالة الخلفية؟

- أول مرة تستخدم الزر، المتصفح يحمّل نموذج ذكاء اصطناعي صغير (يشتغل بالكامل داخل الجهاز، بدون إنترنت بعد أول تحميل، وبدون أي API مدفوع).
- التحميل الأول قد ياخذ حتى دقيقة حسب سرعة النت (لمرة وحدة، يتخزّن بعدها بذاكرة المتصفح).
- بعدها، معالجة كل صورة تاخذ بضع ثوانٍ.
- الناتج: صورة المنتج مفرَّغة الخلفية، موضوعة على خلفية بيضاء نظيفة مع علامة "نقاء" مائية متكررة بخط أخضر فاتح + شعار واضح بالأسفل.

## ⚠️ ملاحظات مهمة

- المنتج يُحفظ مباشرة بنفس جدول `products` بقاعدة البيانات المشتركة — يظهر فوراً بلوحة الإدارة وبتطبيق العميل بمجرد الحفظ.
- الفئة والماركة اختياريتان بهذا التطبيق (تقدر تتركهم فاضيين وتربطهم لاحقاً من لوحة الإدارة، أو تختارهم مباشرة هنا).
- هذا التطبيق **لا يحتاج تسجيل دخول** حالياً (مبني ليكون أداة سريعة داخلية). لو تبي حماية بكلمة سر بسيطة قبل الدخول، قلي وأضيفها.
